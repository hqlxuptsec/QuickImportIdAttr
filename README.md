# QuickImportIdAttr
 这个项目的主要目的是：以AndroidStudio插件的形式解决Android项目中的findViewById，提高开发效率
 1、可以快速导入XML控件；
 2、如果使用ButterKnife，可以直接导入View属性的 相关格式；
本项目是Androidstudio 插件项目，使用时需要先安装插件，目前还没有上传到插件中心，如果有需要可以使用目录下的QuickImportTest.jar安装。
